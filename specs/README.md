# 📘 Guild Agent Framework – Spec Directory

This folder contains the modular design specification for the Guild Agent Framework.

Each file documents a self-contained part of the system. Together, they describe how autonomous agents collaborate to complete complex objectives using shared memory, cost heuristics, and human-in-the-loop coordination.

## Key Documents

- `architecture.md`: Core concepts – agents, guilds, tools, tasks, memory
- `agent-behavior.md`: Agent task flow, cost minimization, tool selection
- `tools.md`: Command-line tools, aides, and tool config schema
- `coordination.md`: Meta-coordination protocol (MCP) and optimization loop
- `objectives.md`: Markdown-based objective planning and prompt design
- `kanban.md`: Task model, statuses, transitions, and event protocol
- `memory.md`: RAG, prompt logging, Qdrant and BoltDB usage
- `configuration.md`: YAML configuration schema and usage
- `examples.md`: Sample guilds and use case walkthroughs
