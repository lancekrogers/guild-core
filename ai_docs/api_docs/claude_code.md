# Claude Code Integration Guide

Claude Code is optimized for agentic workflows and excels at long-form planning, structured reasoning, and code refactoring.

## Key Capabilities

- Maintains long context windows for multi-file reasoning.
- Understands Go package and interface structures.
- Supports code tracing and step-by-step implementation.

## Usage Tips

- Use `.claude/commands/context_prime.md` to set up context.
- Trigger deeper reasoning using `@ultrahink`.
